# Tic-Tac-Toe Game

## Description
A simple implementation of the Tic-Tac-Toe game in C++.

## Technologies Used
- C++

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/Stefanos-gif/tic-tac-toe-game.git
   cd tic-tac-toe-game
   g++ tic_tac_toe.cpp -o tic_tac_toe
   ./tic_tac_toe
